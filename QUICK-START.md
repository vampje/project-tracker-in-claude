# Quick Start Guide

New to this project tracking system? Start here!

---

## 📋 What You Need

- **Time:** 30 minutes to set up, 1 hour/week to maintain
- **Tools:** Any text editor (VS Code, Sublime, Notepad++, or even Notes app)
- **Format:** Markdown (.md files) - easy to read, version control friendly
- **Optional:** AI assistant (Claude, ChatGPT, Copilot) to automate updates

---

## 🚀 Day 1: Setup (30 minutes)

### Step 1: Copy Template Files (5 min)
1. Copy all template files to your own directory
2. Rename them (remove "-template" suffix)
3. Keep the originals as reference

### Step 2: Customize Dashboard (15 min)
1. Open `dashboard.md`
2. Replace example projects with your Q2/Q3 work
3. Add your project names, epics, timelines
4. Set initial status for each project
5. Delete sections you don't need

**Quick wins:**
- Start with just your top 3 projects
- Add more projects as you get comfortable
- Don't try to be perfect - just start!

### Step 3: Create Today's Daily Log Entry (10 min)
1. Open `daily-log.md`
2. Add today's date at the top
3. List what you worked on today
4. Keep it simple - bullets are fine

**Example first entry:**
```
## 2026-03-23 (Monday)

### Work Done
- Set up project tracking system
- Reviewed Q2 priorities
- Sprint planning meeting with team

### Next Actions
- Start tracking daily work in this log
- Update dashboard weekly
```

---

## 📅 Week 1: Build the Habit

### Daily (5 minutes)
**End of day routine:**
1. Open `daily-log.md`
2. Add today's date and what you did
3. Note any blockers or follow-ups
4. Save and close

**That's it!** Don't overthink it.

### Friday (15 minutes)
1. Review your week in `daily-log.md`
2. Add accomplishments to `weekly-updates.md`
3. Copy table rows to your manager's doc
4. Update `dashboard.md` project statuses

---

## 📊 Week 2: Find Your Rhythm

### Try These Improvements

**If daily logging feels like a chore:**
- ✅ Update in the morning instead of evening
- ✅ Keep it open all day, add as you go
- ✅ Use voice-to-text if typing is slow
- ✅ Try an AI assistant to help write entries

**If you're forgetting to update:**
- ✅ Set a calendar reminder (5pm daily)
- ✅ Make it part of your shutdown routine
- ✅ Pin the file to your taskbar/dock
- ✅ Add to your daily checklist

**If it's taking too long:**
- ✅ Use bullet points, not paragraphs
- ✅ Copy/paste ticket numbers instead of typing
- ✅ Skip the formatting (add later if needed)
- ✅ Focus on just the significant work

## 💡 Pro Tips from Experience

### What Successful Users Do

✅ **Update in small chunks** - Don't wait until end of week
✅ **Link everything** - JIRA, docs, Slack - makes it searchable
✅ **Use search** - Cmd+F / Ctrl+F is your friend
✅ **Archive old content** - Keep files manageable
✅ **Share with your team** - Transparency builds trust

### Common Mistakes to Avoid

❌ **Trying to track everything** - Focus on what matters for status
❌ **Perfect formatting** - Done is better than perfect
❌ **Batch updating** - Update as you go, not all at once
❌ **Copy/pasting between files** - Each file has a purpose
❌ **Forgetting to use it** - Set up reminders!

## 🤖 Using with AI Assistants

### What AI Can Do

- **Read your daily log** and generate weekly summaries
- **Draft JIRA epic updates** in executive-friendly language
- **Update your dashboard** based on your conversation
- **Query your logs** to answer "what did I do on X date?"
- **Format links** and maintain consistent structure

### What You Should Still Do

- **Decide what's important** - AI doesn't know your priorities
- **Review AI drafts** - Make sure they're accurate
- **Add context** - AI needs your input on decisions/blockers
- **Own your updates** - Don't blindly trust AI output

### Getting Started with AI

**Option 1: Claude (what I use)**
- Works with VS Code
- Can read/write markdown files directly
- Great for conversational updates
- Costs ~$20/month

**Option 2: ChatGPT**
- Web-based, easy to start
- Copy/paste your logs
- Free tier available
- Pro version ~$20/month

**Option 3: GitHub Copilot**
- Built into VS Code
- Good for auto-completing logs
- ~$10/month

**Option 4: Manual (free!)**
- Totally valid option
- Some people prefer manual control
- No dependencies on AI tools

## 📞 Need Help?

### Troubleshooting

**"I don't know what to track"**
→ Start with: What would your manager ask about? Track that.

**"It's taking too long"**
→ You're tracking too much detail. Use bullets, not paragraphs.

**"I keep forgetting to update"**
→ Set a calendar reminder. Make it a habit like brushing teeth.

**"My manager doesn't use the weekly updates format"**
→ Adapt it! Use for your own reference. Copy key points to their format.

**"I have 20 projects, this won't work"**
→ Focus on top 5-7. Archive the rest. You can't track everything.
